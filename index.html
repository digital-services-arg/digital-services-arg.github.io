Mi servicio principal. Creo páginas web personales para que muestres tu trabajo, proyectos o servicios de forma profesional y atractiva.

Ayuda con Fotos

No soy experto en edición, pero sé lo suficiente para ayudarte. Retoques básicos, ajustes de color, recortes y preparación para web o documentos.

  • Ajustes básicos de color y luz
  • Recortes y composiciones simples
  • Preparación para web o documentos
  • Redimensionamiento para diferentes usos
  • Formato correcto para tu necesidad

Honestidad: No hago retoques profesionales complejos, pero sí puedo ayudarte con lo básico que necesites.

Ayuda con Videos

Edición básica para presentar tu trabajo. Cortes simples, añadir textos, ajustar audio básico y preparar videos para mostrar proyectos o contenido.

  • Cortes y unión de clips
  • Añadir textos y gráficos simples
  • Ajuste básico de volumen
  • Preparación para redes o presentaciones
  • Conversión a formatos adecuados

Honestidad: No hago efectos especiales complejos, pero sí puedo ayudarte a crear videos presentables para mostrar tu trabajo.

Diseño de Documentos
